## Overview / 项目简介

This website is my first documented portfolio project. It preserves the responsive, content-focused Dante theme while replacing the demo material with my own identity and a bilingual English–Simplified Chinese structure.

<span lang="zh-CN">这是我的第一个公开作品集项目。网站保留 Dante 主题的响应式与内容导向设计，同时使用我的个人信息，并加入英文与简体中文双语结构。</span>

## Technology / 技术

- Astro and TypeScript
- Tailwind CSS
- Markdown content collections
- GitHub Actions and GitHub Pages
